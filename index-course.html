algunos conceptos importantes

HTML SEMÁNTICO

Semántico significa que tiene sicnificado, para esto usamos etiquetas, de esta manera le damos un significado a cada parte de la página que estemos creando

Etiqueta "span"

La etiqueta span se usa para aplicar estilo al texto o agrupar elementos en linea. Esto también sirve mucho para cuando queramos estilar solo una parte del texto, envolvemos la parte del testo que queramos estilar con span y luego vamos al css y estilamos

Etiqueta "div"

Cómo su nombre lo indica el div se usa para agrupar elementos-aplicar divisiones lógicas en el contenido de una página web.

Etiqueta "article"

Esta etiqueta article deve tener una información que aunque la quitemos del lugar en el que esté puesto, y lo desplacemos a otro sitio, va a seguir teniendo sentido. Independiente.

Etiqueta "aside"

Esta etiqueta representa una sección de una página que consiste en contenido que está indirectamente relacionado con el contenido principal del documento. Dependiente del documento.

Etiqueta "main"

El main engloba todo el contenido principal de la página, es decir, lo que abarca en el concepto que se hace llamar principal, si hay una pagina de ventas, abarca toda la parte de ventas menos, de la información adicional

Etiqueta "section"

Sirve para abarcar secciones independientes genericas de un doumento, también pueden haber secciones y sub-secciones

Etiqueta "header"

Aunque muchas personas creen que solo se puede usar una vez, realmente el header es una etiqueta que podemos usarla muchas veces para indicar headers incluso de partes de la misma página, al igual que el footer, hay una diferencia entre un header y top-header.

Etiqueta "button"

Esta nos ayuda para crear botones.

Atributo "role"

Hace que los objetos hagan un rol en sí, de manera que podamos hacer que un div, sea un boton, por ejemplo, aunque en programación el div no puede ser boton por cuestiones de semantica. Pero básicamente esa es su funcionalidad.
